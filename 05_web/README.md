# 国旗図鑑 (Web Atlas) 起動・確認マニュアル

このディレクトリ（`05_web`）には、これまでのフェーズで構築した国旗データのプレビュー用Webアプリ（UI/UX最適化済み）が格納されています。

## ローカル環境での起動が必要な理由
本Webアプリは、国旗のメタデータを外部ファイル（`flags_data.json`）から非同期通信（Fetch API）で読み込んでいます。
セキュリティ（CORSポリシー）の制約上、HTMLファイルを直接ダブルクリックしてブラウザで開く（`file://`）だけではデータが読み込まれず、正しく表示されません。

そのため、以下の手順で**ローカルWebサーバー**を一時的に立ち上げて確認を行ってください。

---

## 🚀 起動手順 (Mac / Windows 共通)

### Step 1: ターミナルを開く
*   **Mac**: Spotlight（Command + Space）で「ターミナル（Terminal）」と検索して開きます。
*   **Windows**: スタートメニューから「コマンドプロンプト（cmd）」または「PowerShell」を開きます。

### Step 2: プロジェクトのルートディレクトリに移動
以下のコマンドを入力し、プロジェクトの大元のフォルダへ移動します。（ご自身の環境に合わせてパスを変更してください）

```bash
cd /Users/r-site/Documents/国旗プロジェクト
```

### Step 3: ローカルサーバーを起動
Pythonがインストールされている環境であれば、標準搭載されている簡易サーバーを利用できます。以下のコマンドを入力して実行（Enter）します。

```bash
# Mac または Python3系 の場合
python3 -m http.server 8080

# Windows の場合（python3でエラーが出る場合）
python -m http.server 8080
```

> **成功のサイン**：
> `Serving HTTP on :: port 8080 ...` のようなメッセージが出れば、サーバーが正常に起動しています。

### Step 4: ブラウザでアクセス
Google ChromeやSafariなどのWebブラウザを開き、アドレスバーに以下のURLを入力（またはクリック）してアクセスします。

**👉 [http://localhost:8080/05_web/index.html](http://localhost:8080/05_web/index.html)**

これで、国旗図鑑の全機能（マップフィルタ、色仕様のダウンロード、類似国旗比較等）をブラウザ上でフルにご確認いただけます。

---

## 🛑 終了手順

確認が終わりローカルサーバーを停止したい場合は、サーバーを起動している**ターミナルの画面で `Ctrl + C`（コントロールキーを押しながら Cキー）** を押してください。
即座にサーバーのプロセスが終了し、ターミナルに入力できる状態に戻ります。
